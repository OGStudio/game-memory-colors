Home of "Memory: Colors" game by Opensource Game Studio

Description

The point of the game is to remove matching items.

Below you can find released versions

Memory: Colors 0.3

Web version

Released: 2017-10-10

Features:

  1. Remove matching items
  2. Animate background for a match or a miss
  3. Paint items with fancy patterns
  4. Animate removal of items
  5. Start screen
  6. Finish screen
